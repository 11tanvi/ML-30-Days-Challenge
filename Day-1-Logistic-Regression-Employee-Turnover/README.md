# Day 01 - Logistic Regression (Employee Turnover)

## Problem

Predict whether an employee will leave the company.

## Model

Logistic Regression (with tuning)

## Results

* Accuracy: 85.92%
* Precision: 87.17%
* Recall: 81.6%

## Insights

* Employees with overtime are more likely to leave
* Lower income groups show higher attrition
