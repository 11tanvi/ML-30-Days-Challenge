Day 08: XGBoost on Titanic Dataset

Overview
Built a machine learning model using XGBoost to predict Titanic passenger survival.

Work Done
Handled missing values and encoded categorical features
Performed basic EDA
Trained XGBoost model

Results
Accuracy: ~80%
Precision: ~0.79
Recall: ~0.71

Insight
Model performs well overall but misses some survivors, indicating scope to improve recall.

Next Step
