# Day 6 - Feature Engineering

## Objective
Improve model performance by creating meaningful features instead of changing models.

## Feature Engineering Applied

- Salary_Level 
- Monthly_Income_per_Age
- Satisfaction_Performance
- Income_per_Year
- 
##Model Used
Random Forest Classifier

## Results Comparison

### Before Feature Engineering
- Accuracy: 91.11%
- Precision: 91.05%
- Recall: 89.6%

###After Feature Engineering
- Accuracy: 91.48%
- Precision: 90.47%
- Recall: 91.2%

##Key Insights

- Feature engineering improved **overall accuracy**
- Recall increased → better at identifying employees likely to leave
  
##Learning

Feature engineering plays a crucial role in improving model performance and capturing hidden patterns in data.
