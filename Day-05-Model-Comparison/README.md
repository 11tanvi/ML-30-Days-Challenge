# Day 5 - Model Comparison

Today I compared multiple machine learning models:

- Logistic Regression
- Decision Tree
- Random Forest
- KNN

## Results

Random Forest achieved the highest accuracy (~91%), making it the best model for this dataset.

## Key Learnings

- Ensemble models perform better in many cases
- Model selection is important
- Accuracy alone is not always enough

This step helped me understand which model works best and why.
